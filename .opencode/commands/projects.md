# Projects Command

List known projects for: $ARGUMENTS

## Your Task
1. Scan for directories with `.opencode/` configuration
2. For each project report:
   - Project name and path
   - Instinct count and status breakdown
   - Skill count
   - Command count
   - Last activity date
3. Aggregate totals across all projects

---

**TIP**: Run to discover which projects have active OpenECC configurations.
