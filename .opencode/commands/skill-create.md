# Skill Create Command

Generate skill from: $ARGUMENTS

## Your Task
1. **Analyze git log** — Extract recurring task patterns from commits
2. **Cluster patterns** — Group related actions into skill domains
3. **Generate skill file** — Write `.opencode/skills/<name>/SKILL.md` with:
   - Description and trigger conditions
   - Workflow steps
   - DO/DON'T rules
   - Reference examples from commit history
4. **Validate** — Review for clarity and actionability

---

**TIP**: The best skills encode workflows you've done 3+ times. One-offs don't need skills.
