# Instinct Status Command

View instincts for: $ARGUMENTS

## Your Task
1. Read instinct files from `.opencode/instincts/`
2. List each instinct with:
   - Name and description
   - Source (git history, session learning, manual)
   - Confidence score (based on repetition)
   - Status (active, pending review, deprecated)
3. Summarize total instincts by domain
