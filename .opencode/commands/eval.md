# Eval Command

Evaluate against criteria: $ARGUMENTS

## Your Task
1. **Restate criteria** — List explicit pass/fail conditions
2. **Gather evidence** — Run tests, inspect outputs, review logs
3. **Score each criterion** — PASS / FAIL / PARTIAL with evidence
4. **Overall verdict** — All criteria met? If not, list gaps

## Report Format
```
| Criterion | Status | Evidence |
|-----------|--------|----------|
| ...       | PASS   | ...      |
```

---

**TIP**: Be strict — PARTIAL is not PASS. Gaps must be addressed before completion.
