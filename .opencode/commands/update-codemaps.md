# Update Codemaps Command

Update codemaps for: $ARGUMENTS

## Your Task
1. Run codemap generator or review directory structure
2. Update module/file descriptions for changed areas
3. Add entries for new modules/files
4. Remove entries for deleted modules/files
5. Verify codemap accuracy against actual code

---

**TIP**: Codemaps live in `.opencode/codemaps/` and should be regenerated when the directory structure changes significantly.
